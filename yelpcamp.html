Yelpcamp web application

Node.js, Express, & Mongodb

Yelpcamp is a web application I developed while in Colt Steele's Web Development Bootcamp

Users can register, log in, add new campground sites, post reviews/comments, and edit their own campground entries.

Yelpcamp uses Node.js, Express, Mongodb, Passport, and Mongodb.
